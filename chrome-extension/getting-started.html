Install the Vigil extension

Vigil only assists with forwarding setup in Gmail and Outlook. It does not request inbox read permissions and it does not inspect message bodies.

Load it in Chrome

  1. Open chrome://extensions.
  2. Enable Developer mode.
  3. Click Load unpacked.
  4. Select the chrome-extension folder from this repository.
  5. Pin the Vigil extension so the popup is always one click away.

Connect and configure

  1. Click the Vigil extension icon and sign in with your Vigil account or API key.
  2. If you do not have a watcher yet, open the setup panel and create one there.
  3. Open Gmail or Outlook. The popup can jump directly to each provider's forwarding settings.
  4. Use Assist On Current Page from the setup panel to fill the forwarding address and guide the remaining confirmation steps.
  5. For Gmail, keep the setup panel open while Vigil polls for the forwarding confirmation code.

Backend detection

The extension prefers https://api.vigil.run/api and automatically falls back to a local backend on http://localhost:3001/api when it is available.

Troubleshooting

  1. Reload the extension after pulling changes so the service worker and content scripts refresh.
  2. If Gmail or Outlook already had the settings page open, refresh that tab once so the helper attaches cleanly.
  3. If the setup panel cannot detect your page, use the provider buttons to reopen the forwarding settings and try again.
